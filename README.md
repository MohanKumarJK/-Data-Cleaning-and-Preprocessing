Objective
The objective of this project is to clean and preprocess a raw dataset containing missing values, duplicates, and inconsistent formats. The goal is to transform the data into a clean, structured format suitable for analysis and machine learning.

Tools Used
Python (Pandas)
Microsoft Excel

Dataset
Netflix Movies and TV Shows Dataset

Deliverables
✔️ Cleaned dataset
✔️ Python code for preprocessing
✔️ Summary of changes

Outcome
After cleaning:
Data is consistent and structured
Missing and duplicate values are handled
Dataset is ready for analysis and modeling

Conclusion
This project demonstrates essential data cleaning techniques used in real-world data science workflows. Proper preprocessing improves data quality and ensures accurate analysis.
